문제 15: C언어 - 정수를 역순으로 출력 - 괄호에 연산자 채우기

빈 칸에 연산자를 써서 정수를 역순으로 출력하는 프로그램을 완성하시오.

예를 들어 1234의 역순은 4321이다. 단, 1230처럼 0으로 끝나는 정수는 고려하지 않는다.

#include

int main() {

　

　　　　 int number = 1234;

　　　　　 int divider = 10;

　　　　int result = 0;

　

　　　　 while (number ① 0) {

　

　　　　　　 result = result * divider;

　　　　　　 result = result + number ② divider;

　　　　　　 number = number ③ divider;

　

　　　 }

　

　　　 printf("%d", result);

　

　

}
